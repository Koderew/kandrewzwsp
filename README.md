**Why this project?**

This project was created as an alternative to `zerowidthspace.me`, which frequently fails to load on Firefox and can encounter issues on Chrome when using a VPN (at least in my experience). 


For full transparency, this version is hosted directly via GitHub Pages using the source code in this exact repository, ensuring reliable, open-source, and uninterrupted access across all browsers. The goal is to provide an independent, original-code implementation while keeping the experience and functionality a verbatim copy of the original site.


**Contribute**

If you notice any bugs, layout discrepancies from the original site, or performance issues, feel free to open an issue or submit a pull request! Contributions to keep this implementation accurate, lightweight, and reliable are always welcome.
